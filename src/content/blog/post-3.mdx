The following grainy texture is what I wanted to create for my <a href = "/toolbox"> toolbox page.</a> To do this we just place the noise on top of the color and turn down the opacity.

<Card img = {img1} alt = "Noise"></Card>

**1.** Download this noise.svg image and place it in your public folder.

<Card img = {img2} alt = "Result"></Card>

**2.** In your HTML write...

<Prism lang="html" code={`
<div class = "container">
    <div class = "noise"></div>
</div>

`} />

**3.** In your CSS write...


<Prism lang="css" code={`
.container {
  height: 100%;
  width: 100%;
  background-image: linear-gradient(35deg,
      hsl(320deg 60% 92%) 0%,
      hsl(330deg 100% 93%) 24%,
      hsl(339deg 100% 92%) 36%,
      hsl(354deg 100% 92%) 43%,
      hsl(13deg 100% 89%) 48%,
      hsl(26deg 100% 85%) 52%,
      hsl(36deg 100% 81%) 56%,
      hsl(44deg 100% 77%) 61%,
      hsl(50deg 100% 74%) 67%,
      hsl(55deg 100% 72%) 90%);
}

.noise {
	height: 100%;
	width: 100%;
	background-image: url("noise2.svg");
	opacity: 0.1; //the larger this is the grainer it is 0 to 1 range
}

`} />


Note: To change the gradient colors use the <a href = "https://www.joshwcomeau.com/gradient-generator/">Gradient Generator</a>
